This group project was working on behalf of an existing product or service to identify and solve a user need. My team and I decided to pick Zoom because it is currently an important communication platform. We explored in depth and found some UX issues that seriously needed some improvement.


Methodology

The project was done using an agile scrum methodology, using Asana to plan out weekly sprints with daily standups to recap on project milestones and deliverables. Each team member was responsible for one epic and we would assign tickets to each member as required.
